The prolog program for the solvig the maze is present in maze_solver.pl.
To run and compile:
    1. Run swipl on the terminal.
    2. Now load the maze-solver program: ?- ['maze_solver.pl'].
    3. Now load a Mazedata file with the command: ?- ['Mazedata1.pl'].
    4. For running the shortest path finder, run: ?- ?- shortestPathFinder(src,[dest],dest,X).
        For eg: shortestPathFinder(6,[12],12,X).
    5. Embedded outputs have been commented in maze_solver.pl.
    6. Test case Mazedata files and graphs are also contained in the folder.
